# Lecturer Management App

### Version

v1.0.0

### About The Project

This backend project is created for a lecturer management web application. The REST API was created using undermentioned technologies.

- Java Springboot
- Hibernate ORM
- Hibernate Validator
- Object Mapper
- Lombok
- JWT tokens
- SLF4J, LogBack
- Firebase
- MySQL
